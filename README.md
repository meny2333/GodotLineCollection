# GodotLineCollection

Godot 4.6 游戏关卡集合/启动器。通过 PCK 文件动态加载用户关卡，使用 GAS SDK 云端存档。

本项目与 [godot-line](https://github.com/meny2333/godot-line) 模板配合使用。

## ✨ 特性

- 🎮 **动态加载**：通过 PCK 文件动态加载用户关卡
- ☁️ **云端存档**：使用 GAS SDK 实现云端存档同步
- 🎯 **关卡管理**：内置 PCK 导出与管理工具

## 🚀 快速开始

详细的使用说明请参考 [AGENTS.md](./AGENTS.md)。

如果您想分享您的关卡，可以[将关卡发布到 GodotLineCollection](https://github.com/meny2333/GodotLineCollection)。
