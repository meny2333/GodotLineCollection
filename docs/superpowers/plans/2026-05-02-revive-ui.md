# Revive UI Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 在 CustomGameUI 中添加复活专用 UI 层，玩家死亡且有检查点时显示，提供复活和返回两个选项。

**Architecture:** 单场景面板切换方案，在现有 CustomGameUI.tscn 中添加 ReviveLayer（layer=101），通过显示/隐藏切换复活 UI 和游戏 UI。复活 UI 使用半透明黑色固定样式，不使用自动取色。

**Tech Stack:** Godot 4.6, GDScript, .tscn 场景文件

---

@onready var revive_percent_label: Label = $ReviveLayer/RevivePanel/ProgressContainer/PercentLabel

- [ ] **Step 3: 修改 _ready() 函数**

在 `_ready()` 函数中添加按钮连接和初始化：

```gdscript
func _ready() -> void:
	retry_btn.pressed.connect(_on_revive_pressed)
	back_btn.pressed.connect(_on_back_pressed)
	_hide_revive()
	set_process(true)
	
	UserManager.user_info_updated.connect(_on_user_info_updated)
	
	_apply_circle_avatar(avatar_rect)
	
	# 复活 UI 初始化
	revive_btn.pressed.connect(_on_revive_btn_pressed)
	revive_back_btn.pressed.connect(_on_revive_back_pressed)
	revive_layer.visible = false
```

- [ ] **Step 4: 修改 _process() 函数**

替换现有的 `_process()` 函数：

```gdscript
func _process(_delta: float) -> void:
	var current_state = LevelManager.GameState
	var is_dead = (current_state == LevelManager.GameStatus.Died)
	
	# 复活 UI：死亡 + 有检查点 + 未结束
	var can_revive = is_dead and LevelManager.current_checkpoint != null and not LevelManager.is_end
	
	if can_revive and not _revive_shown:
		_show_revive_ui()
	elif not is_dead and _revive_shown:
		# 复活成功后，隐藏复活 UI
		_hide_revive_ui_silent()
	
	# 更新进度条
	if _revive_shown:
		_update_revive_progress()
	
	# 原有逻辑：显示游戏 UI 时更新数据
	if not _revive_shown:
		if is_dead and not _was_dead:
			_show_revive()
		if not is_dead and _was_dead:
			_hide_revive()
		_was_dead = is_dead
		if _shown:
			_update_ui_data()
```

- [ ] **Step 5: 添加复活 UI 显示/隐藏方法**

添加新方法：

```gdscript
func _show_revive_ui() -> void:
	_revive_shown = true
	revive_layer.visible = true
	ui_layer.visible = false
	_update_revive_progress()

func _hide_revive_ui_silent() -> void:
	"""隐藏复活 UI，不显示游戏 UI（复活成功后调用）"""
	_revive_shown = false
	revive_layer.visible = false

func _hide_revive_ui_show_game() -> void:
	"""隐藏复活 UI，显示游戏 UI（点击返回时调用）"""
	_revive_shown = false
	revive_layer.visible = false
	ui_layer.visible = true
	_update_ui_data()
```

- [ ] **Step 6: 添加进度更新方法**

添加进度条更新方法：

```gdscript
func _update_revive_progress() -> void:
	var percent: int = LevelManager.percent
	revive_percent_label.text = "%d%%" % percent
	revive_progress_bar.value = percent
```

- [ ] **Step 7: 添加复活按钮回调**

添加复活按钮处理函数：

```gdscript
func _on_revive_btn_pressed() -> void:
	"""点击复活：隐藏 UI，复活玩家，继续游戏"""
	_hide_revive_ui_silent()
	if LevelManager.current_checkpoint:
		LevelManager.current_checkpoint.revive()
		if LevelManager.crown > 0:
			LevelManager.is_relive = true
```

- [ ] **Step 8: 添加返回按钮回调**

添加返回按钮处理函数：

```gdscript
func _on_revive_back_pressed() -> void:
	"""点击返回：显示游戏 UI（含重试/返回菜单选项）"""
	_hide_revive_ui_show_game()
```
